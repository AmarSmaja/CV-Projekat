About Me

My passion for web design began in elementary school during a class on HTML pages. Inspired by that lesson, I came up with the idea to create my own blog about basketball and the teams I followed at the time. When choosing a high school, I focused on finding one that offered web development courses. This led me to enroll in the Graphic School in Ilid≈æa, specializing in web design.

Throughout high school, I explored new technologies with every project. This curiosity led me to experiment with JavaScript frameworks like React.js. After graduating, I continued my journey by enrolling at ITAcademy, where I further developed my skills in Java programming and MySQL database management. For my final project, I used Java and MySQL to build a platform for freelance job opportunities.

In my personal projects, I combined my love for basketball and the NBA 2K game to create a website where users can access a database of all players and their cards in the game. Additionally, I explored my passion for gaming by working as an assistant scripter on a GTA San Andreas server and later contributing to a Balkan roleplay server for GTA 5.

My journey reflects my dedication to continuous learning, innovation, and combining my interests with my technical skills to create meaningful projects.
