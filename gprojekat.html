2k Pick'Em React.JS Projekat

Projekt 2K Pick'em razvijen je korištenjem ReactJS library za frontend i JSON baze za pohranu podataka o NBA igračima. Aplikacija omogućava korisnicima pregled i odabir igrača na temelju njihovih statistika i performansi. Maturski Rad

Responzivni dizajn predstavlja ključni pristup u modernom web dizajnu koji osigurava optimalno korisničko iskustvo na različitim uređajima i veličinama ekrana. U mom maturskom radu fokusirao sam se na tehnike implementacije responzivnog dizajna, uključujući korištenje CSS media queries, fleksibilnih grid sistema i responzivnih slika.
